# TO RUN THIS PROJECT

- use git clone <url>
- npm install
- npm start or node pm2.js
- Postman collection was available **docs** Folder
- Configure cron job (crontab -e) for every minute  ```  * * * * * wget  http://127.0.0.1:9000/api/v1/get_jobs   ```
